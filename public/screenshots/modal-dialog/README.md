# Modal/Dialog QA Screenshots

Capture screenshots from `http://localhost:3000/dev-test/modal` and store them in this folder with the filenames below:

- `modal-desktop-open.png`
- `modal-mobile-open.png`
- `modal-form-content.png`
- `modal-closed-backdrop.png`

Recommended viewport presets:

- Desktop: `1440x900`
- Mobile: `390x844`
